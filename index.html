🏰 ยินดีต้อนรับสู่ Hanoi Code Tower

เรื่องราว: พลังงานกลไกของ "หอคอยแห่งฮานอย" ได้หยุดนิ่งลง! มีเพียงพลังแห่งตรรกะคอมพิวเตอร์เท่านั้นที่จะกู้ระบบคืนมาได้ วัตถุประสงค์ของคุณคือการป้อนคำสั่งควบคุมเพื่อย้ายแผ่นจานพลังงานไปยังเป้าหมายอย่างปลอดภัย

สิ่งที่คุณจะได้เรียนรู้และฝึกฝนในหอคอยแห่งนี้:
• ชั้นที่ 1: การประกาศและใช้งาน ตัวแปร (Variables)
• ชั้นที่ 2: การคำนวณด้วย ตัวดำเนินการ (Operators)
• ชั้นที่ 3: การควบคุมเงื่อนไขกฎเหล็ก (Conditions: if-else)
• ชั้นที่ 4: การสั่งการวนซ้ำอัตโนมัติ (Loops: while)

*เกมนี้เปิดให้พิมพ์คำสั่งเองทั้งหมดโดยไม่มีโค้ดสำเร็จรูปให้ก๊อปปี้ จงศึกษาจากโค้ดอ้างอิงให้ถี่ถ้วน!
